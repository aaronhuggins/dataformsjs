DataFormJS [imageGallery] Framework Plugin with Handlebars

Features

  • Shows Overlay with large Image on Thumbnail Click.
  • Handles Left/Right/Escape Keys for the Overlay.
  • Handles Swipe left/right and Tap to close on Mobile Devices.
  • Displays [title] or [alt] of the image with index by default. [title] and [alt] are not required and index can be hidden through CSS if desired.
  • Displays a loading indicator if an image takes longer than 2 seconds to load. The text and timeout can be changed through the API.
  • Overlay style and layout can easily changed with CSS.
  • The code is designed to be small and easy to modify so that it can be easily copied and extended or changed as part of a custom app.
  • Works with all Browsers.

Example Usage and Code

                <!--
    Include plugin [js/plugins/imageGallery.min.js]

    Add attribute [data-image-gallery] and optional [title] or [alt]
    to an <img> or other element for each image.
-->
<img
    src="{{thumbnail}}"
    data-image-gallery="{{image}}"
    alt="{{title}}">

<div
    class="card-image"
    style="background-image: url({{thumbnail}});"
    data-image-gallery="{{image}}"
    title="{{title}}"></div>
